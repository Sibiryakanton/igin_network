Цель создания этого проекта условной социальной сети - презентация навыков, подтверждение опыта применения тех или иных
пакетов. До этого у меня в портфолио были только работы годовалой давности, в которых не отражен прогресс как
разработчика.

На данный момент реализован минимум моделей и кое-какой API с unit-тестами и документацией в Postman, а также
внедрен простой чат с использованием веб-сокетов (отправка сообщений, проверка прав доступа, а также смена
статуса прочтения).

Помимо проделанной работы, планируется найти повод использовать Scrapy для парсинга и Celery для очередей.

--Установка--
Помимо пакетов из requirements.txt для запуска также потребуется докер-контейнер redis. Для его установки нужны команды:
sudo apt install docker.io
docker run -p 6379:6379 -d redis:2.8

Ссылка на документацию к API:
https://documenter.getpostman.com/view/4241636/RWgtTx3g
