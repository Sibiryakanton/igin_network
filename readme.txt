Цель создания этого проекта условной социальной сети - презентация навыков, подтверждение опыта применения тех или иных
пакетов. До этого у меня в портфолио были только работы годовалой давности, в которых не отражен прогресс как
разработчика.

На данный момент реализован минимум моделей и кое-какой API.

Планируется:
1) написать тесты к каждому API-запросу
2) внедрить django-channels для простого мессенджера с использованием веб-сокетов
3) под каким-нибудь предлогом втиснуть Scrapy для парсинга и celery.

--Установка--
Помимо пакетов из requirements.txt для запуска также потребуется докер-контейнер redis. Для его установки нужны команды:
sudo apt install docker.io
docker run -p 6379:6379 -d redis:2.8
