На данном этапе разрабатывается минимальная структура API для демонстрации навыков работы с DRF.
Далее по курсу планируется внедрить django-channels для разработчик чатов с использованием веб-сокетов, celery
(для смс-уведомлений на почту при новых входящих сообщениях) и под каким-нибудь предлогом втиснуть Scrapy для парсинга.


--Установка--
Помимо пакетов из requirements.txt для запуска также потребуется докер-контейнер redis. Для его установки нужны команды:
sudo apt install docker.io
docker run -p 6379:6379 -d redis:2.8
